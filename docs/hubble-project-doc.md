## Chain Name: Hubble

## Type:

Hybrid Rollup (use Optimistic rollup but not support VM, use Zk does not address privacy).

## Tx Cost

0.15 cent (on-chain) and 0.1$ per hour (off-chain)

## Finality

1 block confirm (average 13.5s)

## Consensus

Fraud Proofs

## $ Efficient

No (withdraw from L2 to L1 takes 1 week)

## Average TPS

450 TPS

## Max TPS

2600 TPS
